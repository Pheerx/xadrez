Movimentação das Peças de Xadrez em C

Este projeto simula os movimentos básicos de peças de xadrez utilizando linguagem C. Ele cobre conceitos como:

  • Estruturas de repetição (for, while, do-while)
  • Recursividade
  • Loops aninhados e controle de fluxo (break, continue)

Peças Simuladas

  • Torre - Movimento para direita com recursividade
  • Bispo - Movimento diagonal com recursividade + loops aninhados
  • Rainha - Movimento lateral com recursividade
  • Cavalo - Movimento em "L" com loops complexos

Como Executar

// Compilar
gcc xadrez.c -o xadrez

// Executar
./xadrez

Saída Esperada

O programa imprime no terminal os movimentos de cada peça:

Movimento da Torre (5 casas para a direita):
Direita (5)
Direita (4)
...

Movimento do Cavalo (2 casas para cima e 1 para a direita):
Cima (1)
Cima (2)
Direita (1)

Projeto desenvolvido para fins educacionais, demonstrando lógica e controle de fluxo em C.
