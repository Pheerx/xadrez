Movimentação das Peças de Xadrez em C

Este projeto simula os movimentos das principais peças de xadrez utilizando a linguagem C. Ele demonstra:

  • Uso de estruturas de repetição (for, while, do-while)
  • Funções recursivas
  • Loops aninhados com controle de fluxo

Peças Simuladas

  • Torre - 5 casas para a direita (recursividade)
  • Bispo - Diagonal cima-direita (recursividade + loop aninhado)
  • Rainha - 8 casas para a esquerda (recursividade)
  • Cavalo - Em "L" (2 cima, 1 direita – loops complexos)
